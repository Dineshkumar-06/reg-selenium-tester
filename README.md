# Automation Tester Tool

**Description:**  
A Python-based automation tool that extracts registration form related data from a web application, validates it against predefined requirements, and generates validation reports. Built using **Selenium** for browser automation and **Python** for data validation.

---

## Features
- Automates extraction of registration related data from web forms
- Validates extracted data against requirements
- Handles date formatting, numeric checks, and mandatory field verification
- Generates logs and reports for easy analysis
- Easily extendable to other web form workflows

---

## Tech Stack
- Python 3.x
- Selenium WebDriver
- Pandas (for data handling)
- openpyxl (for handling excel files)
- DiffLib (for difference report generation)
